# Calculator JS (2021)

Meu primeiro projeto em JavaScript! Uma calculadora simples e basica.

## O que eh?

Uma calculadora web simples feita com HTML, CSS e JavaScript vanilla.

## Funcionalidades

- Soma
- Subtracao
- Multiplicacao
- Divisao
- Limpar resultado

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Clique nos botoes de numeros
3. Escolha uma operacao (+, -, *, /)
4. Clique em =
5. Veja o resultado!

## Arquivos

- `index.html` - Estrutura HTML
- `style.css` - Estilos basicos
- `script.js` - Logica da calculadora

## Tecnologias

- HTML5
- CSS3 (basico)
- JavaScript (vanilla)
